## About
**U.C.C** allows you to copy components between game objects with simple process.

## Installation
* Create new folder called **"Editor"** as a subfolder of your **"Scripts"** folder.
* Import **[scripts](https://github.com/iozsaygi/unity-component-copier/tree/master/unity-component-copier/Assets/Scripts/Editor/UCC)** script into folder that you just created.
